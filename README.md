Descrição do tema e da hierarquia.

O projeto tem como tema o gerenciamento de animais, utilizando os conceitos de Programação Orientada a Objetos em Java. Foi implementada uma hierarquia onde a classe Animal atua como superclasse, contendo atributos e comportamentos comuns, como nome, idade e o método fazerSom(). A partir dela, foram criadas subclasses específicas como Cachorro, Gato e Vaca, cada uma com características próprias, como raça ou cor.

A aplicação demonstra o uso de herança, sobrescrita de métodos (@Override) e polimorfismo. No método principal (Main), os objetos das subclasses são armazenados em uma lista do tipo Animal, permitindo que diferentes comportamentos sejam executados de forma dinâmica ao chamar o método fazerSom(). Dessa forma, o sistema evidencia a reutilização de código e a flexibilidade proporcionada pela orientação a objetos.
